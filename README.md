# QR-Code-Generator-🔳 

A fast and simple **QR Code Generator** built with **HTML**, **CSS**, and **JavaScript**.  
This tool allows users to instantly generate QR codes from any text or URL using a free public API.

Perfect for beginners learning API usage, DOM manipulation, and interactive UI design.



## 🚀 Features

### 🔹 Core Features
- Generate QR codes instantly  
- Supports text, links, numbers, and any string  
- Uses a free and reliable QR API  
- Clean and responsive UI  

### 🎨 UI Features
- Minimal and modern design  
- Centered layout  
- Auto-updating QR preview  



## 📂 Project Structure



qr-code-generator/ │ ├── index.html ├── style.css └── app.js




## 💻 Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (Vanilla)** – QR generation logic  

No frameworks required.
